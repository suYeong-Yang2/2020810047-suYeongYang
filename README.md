실행법

2. 터틀이 모두 생성되고 사라진 후 매인 메뉴가 뜨면 '스페이스 바' 키 누르기.

4. 방향키로 장애물을 피해 게임 맵 랜덤 위치에 생성되는 구체(파일 명: soul_~의 모든 파일)를 얻으면 우측 상단 초록색글씨가 1씩 증가. 15가 되면 플레이어의 승리
